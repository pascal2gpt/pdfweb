# Pascal Freyer - Cyberpunk Portfolio

A futuristic, interactive portfolio website designed to showcase AI engineering, policy research, and innovation projects. Built with a "Digital Consciousness" theme featuring neon aesthetics, glassmorphism, and dynamic animations.

## üì¶ Installation & Usage

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/pascal2gpt/pdfweb.git
    ```

2.  **Navigate to the project directory**:
    ```bash
    cd pdfweb
    ```

3.  **Run locally**:
    Since this is a static site, you can open `index.html` directly in your browser, or serve it with a simple HTTP server for the best experience:
    ```bash
    # Python 3
    python3 -m http.server 8080
    ```
    Then open `http://localhost:8080` in your browser.

## üé® Customization

*   **Content**: Edit `index.html` for structure and text.
*   **Translations**: Update the `translations` object in `script.js` to modify or add languages.
*   **Styles**: Modify `styles.css` for custom animations and theme colors. Tailwind classes are used directly in `index.html`.
